# Chapter 6: Keys and Transactions

## Introduction
Bitcoin ownership is proven with keys and digital signatures.

## 6.1 Public/Private Keys Explained Simply
- Public key = receiving address (mailbox).
- Private key = unlocks funds (mailbox key). Never share it.

## 6.2 Peer-to-Peer Transactions (P2P)
1. Create: Alice signs with private key.  
2. Broadcast: Sent to network.  
3. Verify: Nodes validate signature.  
4. Confirm: Miners include in block; irreversible after confirmation.

### Activity: Role-play a Bitcoin Transaction
- Roles: sender, receiver, nodes, miner.  
- Sign, broadcast, verify, include in “block.”
